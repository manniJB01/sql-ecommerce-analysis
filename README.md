# E-Commerce Sales Analysis (SQL)
## 📌 Project Overview
Analyzed 100,000+ orders from a Brazilian e-commerce platform
to uncover revenue trends, top categories, and customer behavior.

## 🛠 Tools Used
- MySQL, MySQL Workbench

## 📊 Key Findings
- Electronics was the top revenue category
- November 2017 had the highest sales (seasonal trend)
- São Paulo accounted for 40%+ of all customers

## 📂 Files
- schema.sql → table creation scripts
- queries.sql → all analysis queries
